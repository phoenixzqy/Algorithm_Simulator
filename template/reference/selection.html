The selection sort is the easiest method of sorting. In this, to sort the d ascending order, the 0th element is compare with all other element. If the 0th element is found to be greater than the compared element then they are interchanged. In this way, after first iteration, the smallest element is placed at 0th position. The procedure is repeated for 1st element and so on.

Analysis:Selecting the smallest element requires scanning all n elements, so this takes n − 1 comparisons and then Swapping or interchanging it into the first position. Finding the next lowest element requires scanning the remaining (n − 1) elements and so on, for
(n − 1) + (n − 2) + ... + 2 + 1 = n (n − 1) / 2=O(n2)
Best Case = O(n2)
Average Case= O(n2)
Worst Case= O(n2)

  • Pros:
    • Selection Sort is simple method.
    • No additional temporary storage is required.
  • Cons:
    • Suitable for small list of elements.
    • General complexity is O (n^2).
